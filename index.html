JannyBot Commands

Below is a list of commands available for the JannyBot. Use these commands in your Discord server to manage user activity and receive notifications.

scruffy/add_user <user>

Adds the specified user to the notification list.

Example: !add_user @UserName

scruffy/set_threshold <number>

Sets the message deletion threshold. When a user deletes more than this number of messages within the specified time period, they will be restricted.

Example: !set_threshold 10

scruffy/whitelist_user <user>

Adds the specified user to the whitelist. Whitelisted users will not be tracked for message deletions.

Example: !whitelist_user @UserName
